## 審査メモ（Review Notes）

```
このアプリは、スマホを振ってスコアを競うシンプルなカジュアルゲームです。

【アプリの使用方法】
1. アプリを起動します
2. ホーム画面の「START」ボタンをタップします
3. 制限時間（10秒）内にスマホを振ります
4. シェイク回数がカウントされ、結果画面に表示されます
5. ランキング画面で過去のスコアを確認できます

【必要な権限】
・モーションセンサー（加速度センサー）: シェイク回数の計測に使用

【特記事項】
・ログインやアカウント登録は不要です
・アプリ内課金はありません
・インターネット接続は不要です
・個人情報の収集は行いません
・すべてのデータはローカルに保存されます
```
